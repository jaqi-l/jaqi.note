    FreeMarker是一款模板语言：即一种基于模板和要改变的数据，并用来生成输出文本的通用工具。

    # assign定义变量

    <#assign userName="jaqi">

    # ${...}输出变量

    <p>${userName}</p>
<!-- 如果为空不执行 -->
<p>${user!}</p>
<!-- !后设置默认值 -->
<p>${user!"jaqi"}</p>

    # 控制语句

    # if

    <#if userName="jaqi">

<#elseif userName="lijiaqi">

<#else>  

</#if>

    # switch

    <#switch value> 
<#case refValue1> 
    ... 
    <#break> 
<#case refValue2> 
    ... 
    <#break> 
    ... 
<#case refValueN> 
    ... 
    <#break> 
<#default> 
    ... 
</#switch>

    # list遍历

    <#list goodsList as item>
  ${item_index+1} 商品名称： ${item.name} 价格：${item.price}
 <!-- 不在最后一项添加换行标签 -->
  <#sep><br><#sep> 
</#list>

    TIP

    1. 循环体+_index返回索引
    2. <#sep>不在第一项之前或最后一项之后显示的内容。
    • 根据name排序
    <#assign userNameArr=[{"name":1},{"name":6},{"name":4}]>
<#list userNameArr?sort_by(["name"])?reverse as user>
   ${user.name}   //  6 4 1
</#list>

    # include引用模板

    <#include "head.html">

    # 常用属性

    • size返回数组长度
    ${(userName?size)}
    • length返回字符串长度
    ${(userName?length)}
    • ??判断变量是否存在,不判断变量的值。
    ${userName??}
    • !!忽略list中的空值
    <#list lists!! as list>
</#list>
    • has_content判断变量是否为空
    ${userName?has_content}
    • ?cNumber类型转字符串
    ${Num?c}
    • 比较运算符

    大于:gt\小于:lt\大于等于:gte小于等于:lte
