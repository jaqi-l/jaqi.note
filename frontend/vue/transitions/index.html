    # transition单组件过渡、动画

    触发条件：DOM的显示隐藏或新增移除

    • name:过渡的类名
      <transition name="transit">
      <p v-show="show">transit</p>
  </transition>

    # transition-group多组件过渡、动画

    触发条件：列表项的修改，插入，删除（列表整体修改不会触发）

    • name:过渡的类名
    • tag:渲染的根元素标签，默认渲染成span标签。*Vue 3.x不再默认渲染
    <transition-group name="list" tag="ul">
    <li v-for="item in list" :key="item" class="list">
      {{ item }}
    </li>
  </transition-group>

    WARNING

    当有相同标签名的元素切换时，需要实则key

    # 过渡、动画的类名

    • 显示过程开始前的状态 .v-enter*Vue 3.x中替换为v-enter-from
    • 显示过程的过渡效果 .v-enter-active
    • 显示过程结束前一刻的状态 .v-enter-to
    • 隐藏过程开始前的状态 .v-leave*Vue 3.x中替换为v-enter-from
    • 隐藏过程的过渡效果 .v-leave-active
    • 隐藏过程结束前一刻的状态 .v-leave-to

    TIP

    v是transition标签的name属性定义的类名

     <transition name="slide-fade">
    <p v-if="show">hello</p>
  </transition>
    • 过度
    .slide-fade-enter-active {
  transition: all .3s ease;
}
.slide-fade-leave-active {
  transition: all .8s cubic-bezier(1.0, 0.5, 0.8, 1.0);
}
.slide-fade-enter, .slide-fade-leave-to{
  transform: translateX(10px);
  opacity: 0;
}
    • 动画
    .slide-fade-enter-active {
  animation: slide-fade-in .5s;
}
.slide-fade-leave-active {
  animation: slide-fade-in .5s reverse;
}
@keyframes slide-fade-in {
  0% {
    transform: scale(0);
  }
  50% {
    transform: scale(1.5);
  }
  100% {
    transform: scale(1);
  }
}

    # 自定义过渡、动画的类名

    • 显示过程开始前的类名v-enter-class

    • 显示过程中的类名v-enter-active-class

    • 显示过程结束前一刻的类名v-enter-to-class

    • 隐藏过程开始前的类名v-leave-class

    • 隐藏过程中的类名v-leave-active-class

    • 隐藏过程结束前一刻的类名v-leave-to-class

    <transition
    name="custom-classes-transition"
    enter-active-class="animated tada"
    leave-active-class="animated bounceOutRight"
  >
    <p v-if="show">hello</p>
  </transition>

    # duration属性

    设置过度、动画的持续时间

    <!-- 设置开始和结束的持续时间 -->
<transition :duration="{ enter: 500, leave: 800 }">
</transition>

    # appear属性

    设置初始渲染的过渡

    <transition
  appear
  appear-class="custom-appear-class"
  appear-to-class="custom-appear-to-class" (2.1.8+)
  appear-active-class="custom-appear-active-class"
>
  <!-- ... -->
</transition>

    # 钩子

    • 初始过程开始前触发before-appear

    • 初始过程开始时触发appear

    • 初始过程开始后触发after-appear

    • 初始过程结束时触发appear-cancelled

    • 显示过程开始前触发before-enter

    • 显示过程开始时触发enter

    • 显示过程开始后触发after-enter

    • 显示过程结束时触发enter-cancelled

    • 隐藏过程开始前触发before-leave

    • 隐藏过程开始时触发leave

    • 隐藏过程开始后触发after-leave

    • 隐藏过程结束时触发leave-cancelled

    <transition
  v-on:before-enter="beforeEnter"
  v-on:enter="enter"
  v-on:after-enter="afterEnter"
  v-on:enter-cancelled="enterCancelled"

  v-on:before-leave="beforeLeave"
  v-on:leave="leave"
  v-on:after-leave="afterLeave"
  v-on:leave-cancelled="leaveCancelled"
>
  <!-- ... -->
</transition>

    # 过度模式

    • 新元素先进行过度，完成之后当前元素过度离开in-out
    • 当前元素先进行过渡，完成之后新元素过渡进入out-in
    <transition name="fade" mode="out-in">
</transition>
