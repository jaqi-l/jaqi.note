# 元素的分类

  • 块元素:主要特征是会产生换行效果，自动与其他元素分离成两行；通常可以作为容器在内部添加其他元素。

h1~h6/hr/ul/ol/p/table

  • 内联元素:不会产生换行效果，会和其他元素并联排列；

b/i/br/img
