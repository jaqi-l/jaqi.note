# 3.13.1基本概念

# 正则：处理字符串的一种规则。

# 正则匹配：验证当前字符串是否符合该规则的子串。

# 正则捕获：获取符合规则的字符。

# 修饰符：

  • /后的表示修饰符
  • iignoreCase忽略大小写
  • gglobal全局匹配
  • mmultiline多行匹配

# 元字符：

  • / /中的内容

# 特殊的元字符

  • \d 0~9任意一个数字

  • \D 非0~9之间的任意字符

  • \w 字母、数字或者下划线

  • \W 非字母、数字下划线

  • \s 匹配任意一个空白字符（包括制表符tab键）

  • \b 匹配边界符（"hello"""为hello的左右边界符）

  • \n 匹配一个换行符

  • \ 转义字符 可以把一个普通字符转换成特殊字符，也可以把特殊的转成普通的 (.小数点)

  • . 表示除了\n之外的任意字符

  • ^ 以某个字符开头

  • $ 以某个字符结尾

  • a|b a或者b任意个

  • [abc] a或b或c任意个

  • [^abc] 除了a/b/c之外的

  • [a-z] a到z任意一个字母（[0-9]→\d）

  • [^a-z]a到z之外的任意字符

# 捕获贪婪性：

  • ()分组

  • ? 表示阻止正则捕获

  • (?:)分组捕获,当前分组只匹配不捕获

  • (?=)前瞻*ES6新增

// 查找以5结尾的a
let str = '1a2a3a4a5a6a'
var n=str.replace(/a(?=5)/,'替换'); //  1a2a3a4替换5a6a
  • (?<=)后瞻*ES2018新增，除chrome以外大部分浏览器都不支持，谨慎使用！
// 查找以5开头的a
let str = '1a2a3a4a5a6a'
var n=str.replace(/(?<=5)a/,'替换'); //  1a2a3a4a5替换6a
  • (?!)反向预查

# 量词元字符

  • * 出现零到多次→{0，}

  • ? 出现零到一次→{0，1}(?取消贪婪性、?:只匹配不捕获、?=)

  • + 出现一次到多次→{1，}

  • {m}出现m次

  • {m,} 出现至少m次

  • {m,n} 出现m到n次

# 普通元字符

正则定义： var arr= [] 或 var arr = new Array();

1.正则字面量：var reg=/\d/g; /d 表示元字符，/表示修饰符

2.new创建：var reg2 = new RegExp("\d","g");

必须是一个符号并且是x： var reg = /^x$/;

# 3.13.2事例：

  1. 以m开头 至少2个n 最多5个n
var reg = /^mn{2,5}$/;
  1. 6个5：
var reg = /^5{6}$/;
  1. 邮编：
var reg = /^[0-9]{6}$/;
  1. 判断用户名字母数字下划线长度范围6~16：
var reg = /^\w{6,16}$/;
  1. 判断用户名字母数字下划线不能以数字开头长度范围6~16：
var reg = /^[a-zA-Z_]\w{5,15}$/;
  1. 至少6位密码
var reg = /^.{6,}$/；
  1. 要求输入网址必须是www.sina.com
var reg = /^www\.sina\.com$/;/
  1. 要求输入 8+9：
var reg = /^8\+9$/;//
  1. 手机号码 以13开头或以15开头：
var reg = /^1[35]\d{9}$/;  
//或
var reg = /^1(3|5)\d{9}$/;
//或
var reg = /^(13|15)\d{9}$/;
  1. 请输入汉字（unicode范围）：
var reg = /^[\u4e00-\u9fa5]+$/;
  1. 18-65：
var reg = /^((1[89])|([2-5][0-9])|(6[0-5]))$/;
  1. 邮箱：用户名@163.com . .. ... ***--.* 要求：用户名 数字 字母 下划线— 不能以.或-开头 不能连续—或.
var reg4 = /^\w+([-.]\w+)*@[a-zA-Z0-9]+([-.][a-zA-Z0-9]+)(\.[a-zA-Z0-9]+)$/;
  1. 书名号里面的标题
/《([^《|》]*)》/
  1. (〔 〕)查询法规标题
/[（\(][a-zA-Z0-9\u4e00-\u9fa5]*?〔{1}.*?〕{1}[a-zA-Z0-9\u4e00-\u9fa5]*?[）\)]/

注意：

  1. 中括号里面的字符绝大多数是普通的字符
  2. 中括号出现的两位数 不是数学的两位数：[18-65]表示1或者8到6或者5

# 3.13.3正则表达式方法

  • test正则检测,返回true/false
var reg = new RegExp(/\s/);
reg.test(str)
  • exec正则捕获，有匹配的值返回该匹配值，否则返回 null。
var reg = new RegExp(/\s/);
reg.exec(str)

  • match返回全部符合的元素

  • search返回符合元素的索引（忽略/g）

  • replace函数的返回值替换符合正则的元素 详见3.9.1
