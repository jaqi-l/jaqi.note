# 5.1请求方式

# 5.1.1get请求

http://localhost/php-day2/1get.php?userName=123456&psw=123456

http://......?K=V&K=V

get请求的表单，一般用于获取数据。

表单两个重要的属性：

method表示提交方法get/post假如写的是get,将来提交表单的时候会通过URL地址？来传递参数

action处理表单的服务器程序

get请求不足：

1.性能不高

2.数据内容不能太大，一般2000字符

优势：便于分享网址

# 5.1.2post请求

post请求也可以把数据传输到服务器上，但是不利用URL,利用HTTP request请求报文头

post一般用于修改服务器上的资源

post请求好处：

1.安全 2.内容不限量

缺点：地址不可以分享

# 5.2ajax

# 5.2.1xhr/open/send

xhr：

高级浏览器：new XMLHttpRequest()

IE6：new ActiveXObject("Micorsoft.XMLHTTP")

兼容：

if (window.XMLHttpRequest) {

xhr = new XMLHttpRequest();

}else{

xhr = new ActiveXObject("Microsoft.XMLHTTP");}

open() //设置代理

// ("请求类型",请求路径",是否异步)

send() //发送请求

()里面的参数是http requset报文头携带的内容

常见异步:setInterval() setTimeout() 所有的事件监听

避免中文乱码：encodeURIComponent()

responseText:接收到的数据

onreadystatechange属性：

0→ open()没有被调用

1→open()被调用

2→头部被服务器接收到

3→开始接收服务器返回的数据

4→完成

if (xhr.readyState == 4) {}

http状态码：

200 ok→ 成功

304 notmodified→已缓存

404 notfound→未找到文件

if(xhr.status>=200 &&xhr.status<300 || xhr.status ==304 ){

content.innerHTML = xhr.responseText;

}else{

content.innerHTML = "服务器内部错误，稍后再试";

}

GET方法配置请求信息：

// 3第一个参数请求类型 get/post，第二个参数请求路径（带参数的写法"php/do2.php?yhm="+yhm），第三个参数是否使用异步。

xhr.open("get","php/1.txt",true);

// 4 发送请求，get写null

xhr.send(null);

}

POST方法配置请求信息：

// 3第一个参数请求类型 get/post，第二个参数请求路径，第三个参数是否使用异步。

xhr.open("post","php/do2.php",true);

// 4设置请求头信息，post必须写这段

xhr.setRequestHeader("Content-Type","application/x-www-form-urlencoded");

// 5发送请求(带参数写法"k=v&k=v")

xhr.send("xingming=along&age=30");

}

# 5.2.3缓存问题及其解决方案

两次get、post请求同一个url携带的参数也一样，即使服务器返回状态码是200 也会被当做304那也缓存，这样会导致后台文件修改，前台内容没有变化

1.加随机数："php/do2.php?x="+Math.random()

2.时间挫：Date.parse(new.Date());

5.3跨域 一个域名的组成：

http://www.abc.com:8080/script/jquert.js

协议+子域名+主域名+端口号+请求资源地址

Js出于安全的考虑，不允许跨域调用的

当协议、子域名、主域名、端口号中任意一个不同时，都算作跨域，

解决方案：

1.后台服务器代理

2.JSNOP(仅支持get请求)

A页面：

<script>

Function jsonp(json){

Alert(json["name"];

}

</script>

<script src= "http://www.bbb.com/jsonp.js "></script>

B页面：

jsonp({'name': 'jaqi','age':24});

3. xhr（XMLHttpRequest），php后端设置

HTML5中的xhr已经支持跨域请求，但是IE10一下不支持。

兼容方案：

header('Access-Control-Allow-Methods:POST,GET');

header('Access-Control-Allow-Credentials:true');
