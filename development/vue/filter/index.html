过滤器只改变展示的形式，不改变原数据，并通过return返回处理后的数据。

  • 全局过滤器
Vue.filter("formatData",function (olddata) {     
    return   newdata  
})
  • 局部过滤器
filters: {
    formatData: function (olddata) {
        return  newdata
    }
}
  • 使用
<p>{{olddata | newdata}}</p>
