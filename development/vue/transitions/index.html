    # transition单组件过渡

    触发条件：DOM的显示隐藏或新增移除

    • name:过渡的类名
      <transition name="transit">
      <p v-show="show">transit</p>
  </transition>

    # transition-group多组件过渡

    触发条件：列表项的修改，插入，删除（列表整体修改不会触发）

    • name:过渡的类名
    • tag:渲染的标签，默认渲染成span标签
    <transition-group name="list" tag="ul">
    <li v-for="item in list" :key="item" class="list">
      {{ item }}
    </li>
  </transition-group>

    # 过渡的类名

    • 显示过程开始前的状态
      .v-enter
    • 显示过程的过渡效果
      .v-enter-active
    • 显示过程结束前一刻的状态
      .v-enter-to
    • 隐藏过程开始前的状态
      .v-leave
    • 隐藏过程的过渡效果
      .v-leave-active
    • 隐藏过程结束前一刻的状态
      .v-leave-to

    TIP

    v是transition标签的name属性定义的类名
